# Python Job Listings Scraper

A lightweight web scraper built with Python to extract job data from the Fake Python Jobs educational site.

## 🚀 Features
- Scrapes Job Title, Company, and Location.
- Extracts unique application links for each posting.
- Exports all data into a clean `job_listings.csv` file.

## 🛠️ Tech Stack
- **Python 3**
- **Requests**: To handle HTTP requests.
- **BeautifulSoup4**: For parsing HTML content.
- **CSV**: For data storage.
